## Quickstart

Install [pnpm](https://pnpm.io/installation) and (Optionally) [Deno](https://docs.deno.com/runtime/getting_started/installation/).

```bash
pnpm run dev
```

## Create a new post

```bash
pnpm post
```

## CI Testing

```bash
pnpm ci:test
```

```bash
pnpm ci:test:quick
```
